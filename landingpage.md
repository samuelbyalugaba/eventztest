# Award-Winning Landing Page Prompt for Lovable

Copy and paste the prompt below into Lovable to generate a powerful, conversion-optimized landing page for the Eventz application.

---

### The Prompt

"Act as a world-class UI/UX designer and frontend developer. Create a high-converting, modern, and visually stunning landing page for **Eventz**, a next-generation Progressive Web App (PWA) designed for event discovery, social engagement, and seamless ticketing.

**The Core Product:**
Eventz is where social media meets event management. It features a TikTok-style immersive feed for event highlights, a robust ticketing system, real-time chat, and professional dashboards for organizers. It's built for the 'mobile-first' generation.

**Design Aesthetic & Visual Identity:**
- **Color Palette:** 
  - Primary: #8A2BE2 (Electric Violet/Purple) - used for primary actions and branding.
  - Background: #FAFAFA (Clean White/Gray) for a spacious, modern feel.
  - Foreground: #1A1A1A (Deep Charcoal) for high-contrast typography.
  - Accents: #00D1FF (Cyan) and #FF3CAC (Hot Pink) for secondary highlights and data visualization.
- **Typography:** Use a clean, sans-serif stack (Inter or System Sans) with a focus on 'Medium' (500) and 'Normal' (400) weights. Headers should be bold and impactful.
- **UI Style:** 'Glassmorphism' accents, soft shadows (`box-shadow`), and rounded corners (`border-radius: 0.75rem`). The UI should feel 'app-like' even on desktop.

**Key Sections to Include:**
1. **Hero Section:** A powerful headline like 'Experience Events Like Never Before.' Include a high-fidelity mockup of the app's immersive video feed and a clear 'Get Started' CTA button in #8A2BE2.
2. **Feature Grid:** Highlight the core pillars:
   - **Immersive Feed:** TikTok-style discovery of local events.
   - **Smart Ticketing:** Secure, instant ticket purchases and scanning.
   - **Social Integration:** Follow organizers, chat with friends, and share highlights.
   - **Organizer Tools:** Professional analytics and event management dashboards.
3. **The 'Experience' Section:** A visual showcase of the mobile interface, emphasizing the PWA nature (Installable, Offline-ready, Fast).
4. **Social Proof/Stats:** Placeholder for 'Active Users,' 'Events Hosted,' and 'Tickets Sold.'
5. **Call to Action Footer:** A final push to 'Join the Revolution' with social media links and a simple newsletter signup.

**Technical Requirements:**
- Use **React** with **Tailwind CSS**.
- Ensure the page is **fully responsive**, with a focus on how it looks on mobile since the app is a PWA.
- Implement subtle entry animations (fade-ins, slide-ups) using `framer-motion` or standard CSS transitions.
- The code should be clean, modular, and easy to integrate with a Supabase backend later.

**The Vibe:**
Energetic, professional, yet social. It should feel like the 'Instagram of Events.' Avoid generic corporate layouts; go for something that feels like a premium tech product."
